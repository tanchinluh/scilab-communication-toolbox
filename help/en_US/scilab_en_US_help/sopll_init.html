sopll_init

Initialize a second order PLL for carrier phase / frequency recovery

Calling Sequence

so = sopll_init(BL, eta)

Parameters

BL:

Loop bandwidth, normalized to sample frequency

eta:

Damping factor

Examples

fs  = 1e6;  // 1 MHz sampling frequency
BL  = 10e3; // 10 KHz loop bandwidth (e.g. ~ 100 samples to converge)
eta = 1;    // Damping factor
so = sopll_init(BL/fs,eta);

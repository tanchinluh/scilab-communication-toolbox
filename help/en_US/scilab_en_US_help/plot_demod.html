plot_demod

Plot the demodulation intermediate results

Calling Sequence

plot_demod(dbg)

Parameters

dbg:

Debug structure as returned by demod_process

Description

This function plots the following figures:

  • PSD before and after down-conversion
  • Demodulation steps (downconversion, carrier recovery, matched filtering, clock recovery) in time and constellation view.
  • Clock recovery convergence

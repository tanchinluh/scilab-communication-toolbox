Filtering and pulse shaping

  • Filters
    • gaussian_fir — FIR approximation of gaussian filter
    • gaussian_pulse — Gaussian pulse
    • ma — Moving average (rectangular) filter
    • rc_fir — FIR approximation of raised cosine filter
    • rc_pulse — Raised cosine pulse
    • srrc_fir — FIR approximation of Square Root Raised Cosine (SRRC) filter
    • srrc_pulse — Square Root Raised Cosine pulse
  • downsample — Downsample the input signal by an integer factor.
  • nrz — NRZ pulse shaping
  • pam — Pulse Amplitude Modulation (PAM)
  • psfilter_init — Creation of a pulse shaping filter
  • psfilter_process — Filter the input signal with a pulse shaping filter
  • symdemap — Maps a symbol sequence to a binary sequence with binary or gray decoding
  • symmap — Maps a binary sequence to a symbol sequence
  • upsample — Upsample the input signal by the factor osf.

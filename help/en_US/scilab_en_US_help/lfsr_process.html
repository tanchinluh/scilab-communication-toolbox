lfsr_process

Compute a random binary sequence with a LFSR register object

Calling Sequence

[x,lfsr] = lfsr_process(lfsr, n)

Parameters

lfsr:

LFSR object (must be created with lfsr_init)

n:

number of bits to generate

x:

Pseudo-random binary sequence
